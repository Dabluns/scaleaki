# Onboarding e Suporte ao Cliente

## Passos para novos usuários
1. Acesse `/auth` e escolha **Criar conta**.
2. Confirme o e-mail (link enviado via `queueConfirmationEmail`).
3. Escolha um plano em `/planos` ou via `/payments/subscription`.
4. Explore o dashboard:
   - Descubraki → ofertas com VSL em formato “reels”.
   - Favoritos/Nichos → salvar oportunidades.
   - Admin (somente perfis autorizados) → gerenciar usuários e ofertas.

## Material sugerido
- **Vídeo curto (2-3 min)**: tour pela landing + Descubraki.
- **Checklist in-app** (pode ser renderizado na home):
  - [ ] Complete o perfil.
  - [ ] Salve 3 ofertas favoritas.
  - [ ] Assista a 2 VSLs.
  - [ ] Ative notificações por e-mail.
- **FAQ** (adicione no frontend):
  - Como recuperar senha?
  - Como trocar de plano?
  - Como entrar em contato com o suporte? (e-mail sugerido: suporte@skaleaki.com)

## Atendimento
- Crie um alias `suporte@skaleaki.com` apontando para o time.
- Registre tickets em ferramenta (Linear/Jira) com tags `bug`, `billing`, `feature`.
- Responda solicitações LGPD usando o inventário documentado em `docs/LGPD_SECURITY.md`.
